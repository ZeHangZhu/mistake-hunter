## 产品简介

**错题猎手**是一款基于Web的智能错题管理系统，采用B/S架构，支持电脑端和移动端同步使用。本产品融合AI技术，帮助学生高效管理错题、制定个性化复习计划，并通过积分激励机制提升学习动力。

### 核心功能

| 功能模块 | 功能描述 |
|---------|---------|
| 📚 错题管理 | 上传错题图片、自动识别、分类管理 |
| 🤖 AI助手 | 智能对话、公式识别、生成相似题目 |
| 📊 统计分析 | 错题统计、掌握程度分析 |
| 🎯 复习计划 | 智能安排复习时间、提醒复习 |
| 🏆 积分系统 | 完成任务获得积分、激励学习 |
| 👨‍🏫 班级管理 | 教师创建班级、查看学生进度 |

## 快速入门

### 系统要求

- **浏览器**：Chrome、Firefox、Safari、Edge 等现代浏览器
- **网络**：需要联网使用AI功能
- **设备**：支持电脑、平板、手机访问

### 第一步：访问系统

打开浏览器，输入系统地址（如：`http://127.0.0.1:8000/`）

### 第二步：注册账号

1. 点击首页的【注册】按钮
2. 填写用户名、邮箱、密码
3. 选择用户类型（学生/教师）
4. 点击【注册】完成账号创建

### 第三步：开始使用

1. **创建学科**：进入【学科管理】，添加第一个学科（如：数学）
2. **添加错题**：点击【添加错题】，上传错题图片
3. **开始复习**：进入【复习计划】，按系统安排进行复习

## 错题管理

### 添加错题

#### 方法一：上传图片

1. 点击【错题本】→【添加错题】
2. 选择学科和分组
3. 点击上传区域，选择错题图片
4. 系统自动识别图片中的文字和公式
5. 填写以下信息：
   - **标题**：错题名称
   - **知识点**：选择或新建相关知识点
   - **难度**：简单/中等/困难
   - **错误原因**：记录出错原因
   - **正确解答**：填写正确答案
   - **解题思路**：记录解题方法
6. 点击【保存】

#### 方法二：手动输入

1. 点击【错题本】→【添加错题】
2. 不选择图片，直接填写题目内容
3. 支持使用 **LaTeX 语法** 输入数学公式
4. 填写其他信息后点击【保存】

### 管理学科

1. 进入【学科管理】页面
2. 点击【新建学科】
3. 填写学科名称，选择主题颜色
4. 点击【保存】

### 管理分组

1. 在学科详情页点击【新建分组】
2. 填写分组名称
3. 可选择父分组，建立层级结构

### 管理知识点

1. 在添加错题时，点击知识点输入框
2. 输入新知识点名称，按回车创建
3. 或从下拉列表选择已有知识点

### 错题筛选与搜索

- **按学科筛选**：点击学科标签
- **按掌握程度筛选**：待复习/已掌握
- **按难度筛选**：简单/中等/困难
- **关键词搜索**：在搜索框输入题目关键词

### 编辑与删除

- **编辑**：点击错题卡片上的【编辑】按钮
- **删除**：点击错题卡片上的【删除】按钮，确认后删除

## AI助手功能

### 智能对话

1. 点击导航栏【AI助手】
2. 创建新对话或选择历史对话
3. 在输入框输入问题，按回车发送
4. AI助手将智能回复，支持数学公式渲染

### 公式识别

上传包含数学公式的图片，AI助手可自动识别并转换为可编辑的LaTeX格式。

### 生成相似题目

1. 在错题详情页点击【生成相似题】
2. AI根据原题知识点和难度生成新题目
3. 可指定：
   - 题目难度（简单/中等/困难）
   - 题目类型（选择题/填空题/解答题）
4. 点击【生成】获取新题目

### 苏格拉底式提问

1. 在错题详情页点击【苏格拉底提问】
2. AI通过引导式提问，帮助理解题目
3. 逐步揭示解题思路，培养独立思考能力

### 对话历史

- 所有AI对话自动保存
- 点击左侧对话列表可查看历史记录
- 支持删除单条对话或清空所有历史

## 积分系统

### 获取积分

| 行为 | 积分奖励 |
|-----|---------|
| 添加错题 | +5 分/题 |
| 完成复习 | +10 分/题 |
| 连续复习7天 | +50 分 |
| 首次使用AI助手 | +20 分 |

### 积分用途

- 查看学习排名
- 解锁高级功能（未来版本）

### 积分中心

1. 点击导航栏【积分中心】
2. 查看当前积分、积分明细
3. 查看积分排行榜

## 复习计划

### 智能复习安排

系统根据艾宾浩斯遗忘曲线，自动安排复习时间：

| 复习次数 | 间隔时间 |
|---------|---------|
| 第1次 | 1天后 |
| 第2次 | 3天后 |
| 第3次 | 7天后 |
| 第4次 | 15天后 |
| 第5次 | 30天后 |

### 开始复习

1. 点击导航栏【今日复习】
2. 系统显示今日需复习的错题列表
3. 点击【开始复习】进入复习模式
4. 查看题目，尝试解答
5. 点击【查看答案】核对
6. 选择掌握程度：
   - **已掌握**：不再安排复习
   - **需复习**：继续安排复习

### 复习统计

1. 进入【复习记录】页面
2. 查看历史复习情况
3. 统计包括：
   - 总复习次数
   - 掌握题目数
   - 复习正确率

### 设置每日复习量

1. 进入【个人中心】
2. 修改【每日复习题目数量】
3. 设置范围：5-50题

## 常见问题

### Q1: 图片上传失败怎么办？

- 检查图片格式（支持 JPG、PNG、GIF）
- 检查图片大小（单张不超过 5MB）
- 检查网络连接
- 尝试刷新页面后重新上传

### Q2: AI助手无法使用？

- 确认已登录账号
- 检查网络连接
- 如长时间无响应，请稍后重试

### Q3: 如何输入数学公式？

使用 LaTeX 语法，例如：
- 行内公式：`$x^2 + y^2 = z^2$`
- 独立公式：`$$\sum_{i=1}^{n} x_i$$`

### Q4: 积分没有增加？

- 确认操作已成功完成
- 积分可能有延迟，稍后刷新查看
- 如问题持续，请联系管理员

### Q5: 忘记密码怎么办？

- 在登录页面点击【忘记密码】
- 输入注册邮箱
- 查收邮件并按指引重置
