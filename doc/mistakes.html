错题管理API

错题管理模块提供了错题的添加、编辑、删除、查看、复习等功能接口，以及学科管理相关接口。

错题列表

GET/mistakes/

错题列表接口，显示用户的所有错题。

请求参数
参数名 类型 必填 描述
subject integer 否 学科ID，用于筛选
group integer 否 分组ID，用于筛选
error_cause string 否 错误原因，用于筛选
mastery_level string 否 掌握程度，用于筛选
search string 否 搜索关键词，用于模糊搜索
sort_by string 否 排序方式，默认为-created_at（按创建时间倒序）
返回结果

返回错题列表页面，包含符合条件的错题。

备注：
  • 需要用户登录后才能访问
  • 支持按学科、分组、错误原因、掌握程度筛选
  • 支持按创建时间、复习次数、难度排序

创建错题

POST/mistakes/create/

创建错题接口，添加新的错题。

请求参数
参数名 类型 必填 描述
title string 是 错题标题
content string 否 错题内容，如果上传图片则自动设置为"图片题目"
solution string 否 解题过程
correct_answer string 否 正确答案
user_answer string 否 用户答案
analysis string 否 题目分析
error_cause string 否 错误原因
custom_error_cause string 否 自定义错误原因
difficulty integer 是 难度等级
subject integer 是 学科ID
group integer 否 分组ID
knowledge_points array 否 知识点ID列表
image file 否 题目图片
solution_image file 否 解题过程图片
correct_answer_image file 否 正确答案图片
user_answer_image file 否 用户答案图片
返回结果

创建成功后重定向到错题详情页面，失败则返回创建页面并显示错误信息。

备注：
  • 需要用户登录后才能访问
  • 支持上传图片，会自动进行OCR识别
  • 创建时会自动设置下次复习时间为1天后

查看错题详情

GET/mistakes/<int:pk>/

错题详情接口，显示错题的详细信息。

请求参数
参数名 类型 必填 描述
pk integer 是 错题ID
返回结果

返回错题详情页面，包含错题的所有信息和复习记录。

备注：
  • 需要用户登录后才能访问
  • 只能查看自己的错题

删除错题

POST/mistakes/<int:pk>/delete/

删除错题接口，删除指定的错题。

请求参数
参数名 类型 必填 描述
pk integer 是 错题ID
返回结果

删除成功后重定向到错题列表页面，失败则返回删除确认页面并显示错误信息。

备注：
  • 需要用户登录后才能访问
  • 只能删除自己的错题

复习错题

POST/mistakes/<int:pk>/review/

复习错题接口，记录复习结果。

请求参数
参数名 类型 必填 描述
pk integer 是 错题ID
result string 是 复习结果，mastered或to_review
notes string 否 复习笔记
notes_image file 否 笔记图片
返回结果

复习成功后重定向到错题详情页面，并显示获得的积分。

备注：
  • 需要用户登录后才能访问
  • 复习会获得积分奖励，包括基础积分、时间奖励和难度奖励
  • 根据复习结果调整下次复习时间

生成复习计划

GET/mistakes/review_plan/

POST/mistakes/review_plan/

生成复习计划接口，根据用户设置生成每日复习计划。

请求参数
参数名 类型 必填 描述
daily_limit integer 否 每日复习数量限制，默认为10
返回结果

返回复习计划页面，包含推荐的复习题目列表。

备注：
  • 需要用户登录后才能访问
  • POST请求会保存用户设置的每日复习数量
  • 根据题目优先级、学科均衡性和知识点均衡性生成计划

创建学科

POST/mistakes/subjects/create/

创建学科接口，添加新的学科。

请求参数
参数名 类型 必填 描述
name string 是 学科名称
color string 否 学科颜色，默认为#007bff
返回结果

创建成功后重定向到学科列表页面，失败则返回创建页面并显示错误信息。

备注：
  • 需要用户登录后才能访问
  • 学科名称不能重复
  • 会根据学科名称自动设置排序值

删除学科接口，删除指定的学科。

请求参数

备注：

  • 如果学科下有错题，无法删除
