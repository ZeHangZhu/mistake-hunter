用户管理API

用户管理模块提供了注册、登录、注销、密码重置等功能接口。

注册

POST/register/

用户注册接口，创建新用户账号。

请求参数
参数名 类型 必填 描述
username string 是 用户名
email string 是 邮箱地址
password string 是 密码
confirm_password string 是 确认密码
返回结果

注册成功后重定向到登录页面，失败则返回注册页面并显示错误信息。

备注：
  • 用户名不能重复
  • 邮箱不能重复
  • 两次输入的密码必须一致

登录

POST/login/

用户登录接口，验证用户身份并创建会话。

请求参数
参数名 类型 必填 描述
login string 是 用户名或邮箱
password string 是 密码
返回结果

登录成功后重定向到仪表盘页面，失败则返回登录页面并显示错误信息。

备注：
  • 支持使用用户名或邮箱登录
  • 密码验证失败会显示错误信息

注销

GET/logout/

用户注销接口，销毁用户会话。

请求参数

无

返回结果

注销成功后重定向到登录页面。

忘记密码

POST/forgot-password/

忘记密码接口，发送密码重置邮件。

请求参数
参数名 类型 必填 描述
email string 是 注册时使用的邮箱地址
返回结果

提交成功后显示邮件发送成功信息，失败则显示邮箱未注册错误。

备注：
  • 系统会生成密码重置链接并发送到用户邮箱
  • 重置链接包含用户ID和安全令牌

重置密码

POST/reset-password/<uidb64>/<token>/

重置密码接口，通过邮件中的链接重置用户密码。

请求参数
参数名 类型 必填 描述
uidb64 string 是 用户ID的base64编码
token string 是 安全令牌
password string 是 新密码
confirm_password string 是 确认新密码
返回结果

密码重置成功后重定向到登录页面，失败则显示错误信息。

备注：
  • 重置链接有效期有限
  • 两次输入的新密码必须一致

仪表盘

GET/dashboard/

用户仪表盘接口，显示用户学习数据概览。

请求参数

无

返回结果

返回仪表盘页面，包含以下数据：

  • 总错题数
  • 待复习错题数
  • 已掌握错题数
  • 学科数量
  • 最近添加的5道错题
备注：
  • 需要用户登录后才能访问

公告详情

GET/announcement/<int:announcement_id>/

公告详情接口，显示系统公告内容。

请求参数
参数名 类型 必填 描述
announcement_id integer 是 公告ID
返回结果

返回公告详情页面，显示公告内容。

备注：
  • 目前系统内置了3个固定公告
  • 公告内容使用Markdown格式
