AI辅导API

AI辅导模块提供了复习计划、苏格拉底式对话、相似题生成等功能接口。

复习计划

GET/ai/review-plan/

复习计划接口，显示用户的复习计划。

请求参数

无

返回结果

返回复习计划页面，包含待完成的复习计划。

备注：
  • 需要用户登录后才能访问
  • 只显示当前日期及之前的未完成计划

苏格拉底式对话

GET/ai/socratic/<int:mistake_id>/

POST/ai/socratic/<int:mistake_id>/

苏格拉底式对话接口，与AI进行交互式对话。

请求参数
参数名 类型 必填 描述
mistake_id integer 是 错题ID
content string 否 用户输入的对话内容，仅POST请求需要
返回结果

返回对话页面，包含对话历史和AI回复。

备注：
  • 需要用户登录后才能访问
  • 只能对自己的错题进行对话
  • 当前AI回复为占位内容，后续会接入真实API

结束对话

GET/ai/socratic/<int:session_id>/end/

结束对话接口，结束当前的苏格拉底式对话。

请求参数
参数名 类型 必填 描述
session_id integer 是 对话会话ID
返回结果

结束对话后重定向到对应错题的详情页面。

备注：
  • 需要用户登录后才能访问
  • 只能结束自己的对话

生成相似题

GET/ai/similar/<int:mistake_id>/generate/

POST/ai/similar/<int:mistake_id>/generate/

生成相似题接口，基于原题生成相似的题目。

请求参数
参数名 类型 必填 描述
mistake_id integer 是 错题ID
返回结果

生成成功后重定向到相似题列表页面，失败则返回生成页面并显示错误信息。

备注：
  • 需要用户登录后才能访问
  • 只能基于自己的错题生成相似题
  • 当前生成的相似题为占位内容，后续会接入真实API

查看相似题

GET/ai/similar/<int:mistake_id>/

查看相似题接口，显示基于原题生成的相似题。

请求参数
参数名 类型 必填 描述
mistake_id integer 是 错题ID
返回结果

返回相似题列表页面，包含基于原题生成的所有相似题。

备注：
  • 需要用户登录后才能访问
  • 只能查看基于自己错题生成的相似题
