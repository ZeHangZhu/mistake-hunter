AI聊天API

AI聊天模块提供了与AI助手进行对话的功能接口，包括创建对话、发送消息、获取对话历史等。

聊天首页

GET/ai/chat/

聊天首页接口，显示AI聊天界面。

请求参数

无

返回结果

返回聊天首页，包含对话列表和消息输入界面。

聊天流

POST/ai/chat/chat-stream/

聊天流接口，发送消息并获取AI的流式回复。

请求参数
参数名 类型 必填 描述
message string 是 用户发送的消息内容
conversation_id integer 是 对话ID
返回结果

返回流式响应，包含AI的回复内容。

data: {"choices":[{"delta":{"content":"你好"},"index":0,"finish_reason":null}]} data: {"choices":[{"delta":{"content":"！"},"index":0,"finish_reason":null}]} data: [DONE]
备注：
  • 使用Server-Sent Events (SSE) 协议返回流式响应
  • 会将用户消息和AI回复保存到对话历史中
  • 如果对话标题为"新对话"，会自动根据用户消息设置对话标题

获取对话列表

GET/ai/chat/conversations/

获取对话列表接口，返回所有对话的基本信息。

请求参数

无

返回结果

返回JSON格式的对话列表，包含每个对话的ID、标题、创建时间和更新时间。

{ "conversations": [ { "id": 1, "title": "数学问题", "created_at": "2026-02-20T10:00:00Z", "updated_at": "2026-02-20T10:30:00Z" }, { "id": 2, "title": "新对话", "created_at": "2026-02-20T11:00:00Z", "updated_at": "2026-02-20T11:00:00Z" } ] }

创建对话

POST/ai/chat/conversations/create/

创建对话接口，创建一个新的对话。

请求参数

无

返回结果

返回JSON格式的新对话信息，包含对话ID、标题、创建时间和更新时间。

{ "id": 3, "title": "新对话", "created_at": "2026-02-20T12:00:00Z", "updated_at": "2026-02-20T12:00:00Z" }

获取对话消息

GET/ai/chat/conversations/<int:conversation_id>/

获取对话消息接口，返回指定对话的所有消息。

请求参数
参数名 类型 必填 描述
conversation_id integer 是 对话ID
返回结果

返回JSON格式的消息列表，包含每条消息的角色和内容。

{ "messages": [ { "role": "user", "content": "什么是二次函数？" }, { "role": "assistant", "content": "二次函数是形如f(x) = ax² + bx + c的函数，其中a≠0。" } ] }

删除对话

DELETE/ai/chat/conversations/<int:conversation_id>/delete/

删除对话接口，删除指定的对话。

请求参数
参数名 类型 必填 描述
conversation_id integer 是 对话ID
返回结果

返回JSON格式的删除结果。

{ "success": true }

获取最近错题

GET/ai/chat/mistakes/

获取最近错题接口，返回用户最近添加的错题。

请求参数

无

返回结果

返回JSON格式的最近错题列表，包含错题ID、标题、内容、学科名称、创建时间和图片URL。

{ "mistakes": [ { "id": 1, "title": "二次函数最值问题", "content": "求函数f(x) = x² - 4x + 3的最小值", "subject_name": "数学", "created_at": "2026-02-20T09:00:00Z", "image_url": null } ] }
备注：
  • 最多返回20条最近错题
  • 如果错题有图片，会返回第一张图片的URL和OCR识别结果
